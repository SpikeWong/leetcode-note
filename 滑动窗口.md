[53. 最大子序和](https://leetcode-cn.com/problems/maximum-subarray/)

```python
解法一(动态规划)：
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        pre = 0
        result = nums[0]
        for i in nums:
            pre = max(pre + i, i)
            result = max(pre, result)
            
        return result
        
时间复杂度: O(n)
空间复杂度: O(1)

解法二(线段树)：
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        _,_, result, _ = self.helper(0, len(nums)-1, nums)
        return result

    def helper(self, l, r, nums):
        if l == r:
            return nums[l], nums[l], nums[l], nums[l]
        
        mid = (l + r) >> 1
        lsum1, rsum1, msum1, isum1 = self.helper(l, mid, nums)
        lsum2, rsum2, msum2, isum2 = self.helper(mid+1, r, nums)
        lsum = max(lsum1, isum1+lsum2)
        rsum = max(rsum2, isum2+rsum1)
        isum = isum1 + isum2
        msum = max(msum1, msum2, rsum1+lsum2)
        return lsum,rsum, msum, isum
        
时间复杂度：假设我们把递归的过程看作是一颗二叉树的先序遍历，那么这颗二叉树的深度的渐进上界为O(logn)，这里的总时间相当于遍历这颗二叉树的所有节点，故总时间的渐进上界是O(∑i=1logn2i−1)=O(n) ，故渐进时间复杂度为 O(n)。
空间复杂度：递归会使用O(logn) 的栈空间，故渐进空间复杂度为O(logn)

```



[26. 删除排序数组中的重复项](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-array/)

```python
解法一:
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if len(nums) == 0:
            return 0
        i = 1
        for j in range(1, len(nums)):
            if nums[j - 1] != nums[j]:
                nums[i] = nums[j]
                i += 1
        return i
        
时间复杂度：O(n)
空间复杂度：O(1)

```



[977. 有序数组的平方](https://leetcode-cn.com/problems/squares-of-a-sorted-array/)

```
解法一:
class Solution:
    def sortedSquares(self, A: List[int]) -> List[int]:
        length = len(A)
        ans = [0] * length
        i = 0
        j = length - 1
        index = length - 1
        while i <= j:
            if abs(A[i]) > abs(A[j]):
                ans[index] = A[i] * A[i]
                i += 1
            else:
                ans[index] = A[j] * A[j]
                j -= 1
            index -= 1
            
时间复杂度: O(n)
空间复杂度：O(n)


解法二：
class Solution:
    def sortedSquares(self, A: List[int]) -> List[int]:
        index = self.partition(A, 0, len(A)-1)
        if A[index] < 0:
            index = index  + 1

        nums1, nums2 = self.power(A[:index]), self.power(A[index:])
        return self.merge(A, index)

    def partition(self, nums, l, r):
        while l <= r:
            mid = l + (r - l) // 2
            if nums[mid] * nums[mid-1] <= 0:
                return mid
            if nums[mid] > 0:
                r = mid - 1
            else:
                l = mid + 1
        return 0
        
    def merge(self, nums, index):
        n = len(nums)
        ans = []
        i, j = index - 1, index 
        while i >= 0 or j < n:
            if i < 0:
                ans.append(nums[j] * nums[j])
                j += 1
            elif j == n:
                ans.append(nums[i] * nums[i])
                i -= 1
            elif nums[i] * nums[i] < nums[j] * nums[j]:
               ans.append(nums[i] * nums[i])
               i -= 1
            else:
               ans.append(nums[j] * nums[j])
               j += 1
            
        return ans
        
        
时间复杂度: O(n)
空间复杂度: O(n)

```



[15. 三数之和](https://leetcode-cn.com/problems/3sum/)

```
解法一（暴力法）:
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        result = []
        self.helper(nums, 0, 3, result, [])
        return result

    def helper(self, nums, target, number, result, temp):
        if number == 1:
            if -target in nums:
                b = temp + [-target]
                b.sort()
                if b not in result:
                    result.append(b) 
            return 

        for i, v in enumerate(nums):
            self.helper(nums[i+1:], target+v, number - 1, result, temp + [v])
        return 
        
        

解法二（）：
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        nums.sort()
        result = []
        for first in range(n):
            if first > 0 and nums[first-1] == nums[first]:
                continue
            third = n - 1
            
            for second in range(first+1, n):
                if second > first + 1 and nums[second] == nums[second-1]:
                    continue
                while second < third and nums[second] + nums[third] > -nums[first]:
                    third -= 1
                if third == second:
                    break
                if nums[second] + nums[third] == -nums[first]:
                    result.append([nums[first], nums[second], nums[third]])
        return result
            
时间复杂度: o(n)
空间复杂度: o(n)
```
